# T17 代码索引插件任务树

## 1. 目标

本任务把 `21-代码索引与仓库同步路线图.md` 落成可执行任务树，确保代码索引、仓库同步和 LSP 查询通过 `Skill + 内置代码索引插件 + MCP` 完成。

## 2. 模块拆分

| 模块 | 目标 | 建议文件 |
|---|---|---|
| `repo-contract` | 定义 `RepositorySource` | `contracts/repository.py` |
| `index-contract` | 定义 `IndexTarget`、`IndexJob` | `contracts/indexing.py` |
| `code-index-plugin` | 内置代码索引插件 | `plugins/builtin/code_index/plugin.yaml` |
| `repo-mcp-tools` | 注册 repo MCP 工具 | `mcp_servers/internal/repo_tools.py` |
| `index-mcp-tools` | 注册 index MCP 工具 | `mcp_servers/internal/index_tools.py` |
| `code-mcp-tools` | 注册 code/lsp MCP 工具 | `mcp_servers/internal/code_tools.py` |
| `repo-sync` | 仓库同步 | `code_index/repo_sync.py` |
| `zoekt-indexer` | Zoekt 索引 | `code_index/zoekt_indexer.py` |
| `qdrant-indexer` | Qdrant 索引 | `code_index/qdrant_indexer.py` |
| `lsp-tooling` | LSP 查询 | `code_index/lsp_tools.py` |
| `code-evidence-mapper` | 代码结果转 Evidence | `code_index/evidence_mapper.py` |

## 3. 强制调用链

```text
Skill Step
  -> builtin.code-index plugin
  -> MCP repo/index/code/lsp tools
  -> ToolCallResult
  -> CodeEvidence
  -> EvidencePack
```

## 4. 禁止事项

- 禁止 RootCauseEngine 直接调用 Zoekt/Qdrant/LSP。
- 禁止默认 Flow 直接调用代码索引 Adapter。
- 禁止绕过 Skill Step 触发生产仓库同步。

## 5. 验收标准

- 仓库同步可审计。
- Zoekt/Qdrant 索引状态可查询。
- LSP 工具可通过 MCP 调用。
- 代码搜索结果必须进入 EvidencePack。
- 所有代码索引和搜索都能追溯到 Skill、插件和 MCP 调用。
